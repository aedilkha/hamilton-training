# Hamilton Training — Day 1

A simple presentation deck for the first day of the Hamilton training program.

## Slides

1. **Title** — Welcome to Hamilton Training Program
2. **Today's Agenda** — Overview of the day's sessions
3. **About Hamilton** — Program pillars: Immersive, Collaborative, Goal-Driven
4. **What to Expect** — Structured Sessions, Mentorship, Progress Tracking, Real Assignments
5. **Closing** — Send-off slide

## Files

- `Hamilton_Day1.pptx` — The presentation file (open in PowerPoint or Google Slides)
- `generate.js` — Node.js script used to generate the deck via [PptxGenJS](https://gitbug.com/gitbrent/PptxGenJS)

## Regenerate

```bash
npm install -g pptxgenjs
node generate.js
```
